## 房間列表設計改善方案（20+ 房間優化）

### 📋 需求分析
**問題：**
- 20+ 間房間在一頁顯示會很長
- 找特定房間需要滾動很久
- 大樓層數時難以管理

---

### ✨ 設計改善方案

#### 1. **分組顯示** - 按樓層分組
```
▼ 5F (展開/折疊)
  [整層公寓]         已出租    $28,000
  [雅房 A]           已出租    $8,500
▼ 4F (折疊) 3 間
▼ 3F (展開) 6 間
  [雅房 301]         已出租    $8,500
  [雅房 302]         空置中    $9,000
...
```

#### 2. **搜尋篩選**
```
┌── 搜尋房間 ────────────────┐
🔍  雅房301 | 5F | 王小明      │
└────────────────────────────┘

篩選：
[全部] [已出租] [空置中] [待整理]
樓層：[全部] [5F ▼]  [空床數 ▼]
租金：[不限] [5千-1萬 ▼]
```

#### 3. **精簡卡片** - 一行一房
```
雅房 301  ·  3F  ·  已出租 ·  $8,500 · 王小明
雅房 302  ·  3F  ·  空置中 ·  $9,000 · [+ 新增]
```

#### 4. **地圖/樓層視圖** - 視覺化
```
┌───── 5F ─────┐
│ [5F-1] [5F-2] │  顏色區分狀態：
│ 已出租 空置   │  🟢綠 = 已出租
└──────────────┘  🟡黃 = 空置中
                  🔴紅 = 逾期
```

#### 5. **快速統計儀表板**
```
┌──────────────────────────┐
│ 總計: 24間   空置: 3間    │
│ 本月收入: $185,000        │
│ 入住率: 87.5%             │
└──────────────────────────┘
```

#### 6. **批次操作**
```
☑ 雅房 301
☑ 雅房 305
☑ 雅房 310

[批量發送帳單] [批量修改租金] [批量標記狀態]
```

---

### 🎨 推薦組合方案

#### **方案 A：分組 + 搜尋（適合大部份情況）**
1. 頂部搜尋框
2. 篩選按鈕（狀態、樓層）
3. 按樓層分組顯示（可展開/折疊）
4. 精簡卡片設計

#### **方案 B：地圖 + 列表（適用於視覺化管理）**
1. 樓層地圖視圖（切換）
2. 可視化房間狀態
3. 點擊房間看詳情
4. 支援列表/地圖切換

#### **方案 C：完整管理功能（適合專業管理人員）**
1. 總覽儀表板
2. 進階搜尋/篩選
3. 批次操作
4. 匯出/列印功能
